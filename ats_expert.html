ATS-Expert AI

AI-Powered Resume Analyzer

Built with Gemini API and Firestore, this app evaluates resumes against job descriptions and gives real-time ATS score and suggestions.

Live Demo View Code

Features

  • 🧠 Gemini API for resume parsing & job matching
  • 📊 ATS score calculation & keyword suggestions
  • 🔒 Google OAuth integration
  • ☁️ Real-time data with Firestore

Tech Stack

  • Python (Streamlit)
  • Gemini API
  • Firebase Firestore
  • Google OAuth 2.0

My Role

I built the entire project from scratch including integration with Gemini API, designing the user interface, configuring Google login, and setting up Firestore for real-time storage.

Challenges & Learnings

Handling Google OAuth redirect mismatches and optimizing Gemini API requests for better relevance were key challenges. I also learned how to improve resume-job alignment using keyword-based scoring.
